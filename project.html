Project overview

PVVP: Visual Attention Research Platform

Topic: Platforma pre výskum vizuálnej pozornosti

Acronym: PVVP

Team Leader: Ing. Peter Kapec, PhD.

What We're Building

The project aims to build a cloud platform for analyzing and visualizing eye-tracking data to evaluate saliency models. Designed for researchers and UX professionals, the system supports comparative model evaluation with metrics, graphs and visualizations

Used Technologies

Vue.js
FastAPI
PostgreSQL
Docker
Kubernetes
